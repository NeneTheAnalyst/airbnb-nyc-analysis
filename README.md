# Airbnb NYC Private Room Analysis

This project analyzes Airbnb listing data in New York City, focusing on private rooms.

## 📁 Files

- `Exploring Airbnb Market Trends.ipynb`: The main notebook with the data cleaning and analysis.

## 📊 What This Project Does

- Loads and merges Airbnb datasets
- Finds the earliest and most recent review dates
- Counts the number of private room listings
- Calculates the average price of listings
- Combines the summary statistics into a single DataFrame
